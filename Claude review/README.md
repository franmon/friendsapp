# BachelorApp 🎉

App móvil para organizar despedidas de soltero. Construida con React Native + Expo + Supabase.

## Stack
- React Native + Expo SDK 54
- Supabase (Auth, DB, Storage, Realtime)
- Expo Router (navegación)
- TypeScript

## Módulos
- Auth + Grupos con código de invitación
- Perfiles + Modo discreción
- Calendario de actividades + Notificaciones push
- Gestión de gastos + Liquidación automática
- Fotos compartidas + Mapa + Cápsula del tiempo
- Documentos de viaje (boarding passes, hotel)
- Votaciones, Timeline, Contador regresivo
- Collage automático de fotos
